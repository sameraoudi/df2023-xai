# df2023-xai
This repository contains the official source code for the paper "Explainable Deep Learning for Image Forgery Detection: Interpreting Manipulation Cues in DF2023". It addresses the "black-box" problem in digital forensics by combining high-performance segmentation models (SegFormer-B2, U-Net R34) with post-hoc Explainable AI (XAI) methods.
